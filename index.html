Programming for Artists - Machine Learning Demo

Using PoseNet model to estimate poses and draw skeleton lines and write SEM on pose keypoints. Running on an underpowered Rasberry Pi 4 so the framerate is choppy. To try the demo on your phone, go to https://buldendi.github.io/poses/ Made with p5.js and ml5.js and demo code from ml5.js documentation site. The original PoseNet model was ported to TensorFlow.js by Dan Oved.
